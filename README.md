# GroupDocs.Merger - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Merger](https://docs.groupdocs.com/merger/) static documentation.

## About GroupDocs.Merger

[GroupDocs.Merger](https://products.groupdocs.com/merger/) is a document manipulation SDK that enables you to merge, split, reorder, swap, and remove pages from documents in various formats.

Available for .NET, Java, Node.js, and Python.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/merger/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
