[**GroupDocs.Merger for Python via .NET**](https://products.groupdocs.com/merger/python-net) provides a complete set of security operations that let you protect and manage document access. You can add, update, or remove open passwords on documents in any supported format, and apply fine-grained permission restrictions to PDF files.

## Available Operations

- [Add document password]({{< ref "merger/python-net/developer-guide/security-operations/add-document-password.md" >}}) — encrypt a document with a new password using `AddPasswordOptions`.
- [Add password to PDF with permissions]({{< ref "merger/python-net/developer-guide/security-operations/add-password-to-pdf-with-permissions.md" >}}) — protect a PDF and restrict specific permissions using `PdfSecurityOptions`.
- [Check document password protection]({{< ref "merger/python-net/developer-guide/security-operations/check-document-password-protection.md" >}}) — determine whether a document has an open password set using `is_password_set()`.
- [Remove document password]({{< ref "merger/python-net/developer-guide/security-operations/remove-document-password.md" >}}) — decrypt a password-protected document using `remove_password()`.
- [Update document password]({{< ref "merger/python-net/developer-guide/security-operations/update-document-password.md" >}}) — replace an existing password with a new one using `UpdatePasswordOptions`.
